# TodayLunch
점심메뉴 추천 사이트 

매번 점심을 고르는데 많은 시간을 소비한다. 이를 개선하고 즐거움을 주는 사이트 개발.

## 주요 기능
1. 메뉴를 랜덤으로 추천해주는 룰렛
2. 음식을 토너먼트 형식으로 골라주는 음식 이상형 월드컵
3. 3개의 옵션을 선택하면 그 날의 음식을 추천해주는 음식 추천 서비스
4. 음식을 검색하고 인근 식당을 추천해주는 지도검색 서비스
5. 일반 음식 메뉴추천
6. 인근 맛집탐방

* 로그인, 회원가입 기능
